# ausc4p

A Universal Smart Computer for Professionnals

The product is pocket-sized, has a battery killer switch on all wires, a power switch on all wires, a microphone switch on all wires, a front camera switch, a back camera switch, a wifi switch, a GPS switch, a gyroscope switch on all wires, an accelerometer switch on all wires, a NFC switch and a Bluetooth switch. All these switches shall be material and, when possible, shall act on both data and current circuits because they control the access to a critical peripheral. The cited peripherals are critical because they collect and/or process user data.

Each critical peripheral shall have a gigabit ethernet interface and its own kernel in order to provide real-world separation. The network peripherals are a standard Wifi interface, a FPGA-based Wifi interface, a FPGA-based Bluetooth interface and maybe a FPGA-based USB Type C interface for physical layer research, FPGA chips are made for development on how to provide long lasting support because they allow deployment of new circuits at each restart of the FPGA chips.

The device shall be built to last 20 years.

2 PCI express buses on 2 different boards with 1 USB 4.0 connector each will allow developers to plug a GSM modem in an isolated board and in the main board to compare. Multiple screens can be plugged if there is enough power to drive it, depending on system load.
